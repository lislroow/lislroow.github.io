1. vm 환경 구성

1.1 vmware 환경 구성

1) vmware workstation 설치

  • 다운로드: broadcom

2) vmware 설정: NAT 설정

  • Edit → Virtual Network Editor → NAT Type: VMnet8 선택 → Change Settings 클릭
    • Subnet IP: 172.28.200.0
    • Subnet mask: 255.255.255.0

3) rocky linux 10 다운로드

  • Rocky-10.1-x86_64-dvd1.iso (8.64 GB)

4) vm 생성

  • (탐색기) C:\vm\study 디렉토리 생성
  • (vmware) File → New Virtual Machine
    • Typical 선택
    • I will install the operating system later 선택
    • Linux 선택
    • Location: C:\vm\study 디렉토리, Virtual Machine Name: develop.el10
    • 20 GB → (Finish)
  • (vmware) develop.el10 선택
    • Memory: 4 GB
    • Processor: 4 (1 / 4)
    • CD/DVD (IDE) → (Use ISO image file) Rocky-10.1-x86_64-dvd1.iso

1.2 Guest OS 설치

  • develop.el10 Guest OS 시작
  • Rocky Linux 설치 진입 → Locale 선택: 한국어, 한국 선택

1) Network & Host Name 설정

  • Ethernet(ens160) → 활성화
  • configure →
    • IPv6 Settings → Method → (Automatic > Disabled)
    • IPv4 Settings →
      • Method → (Automatic > Manual)
      • Address Add →
        • Address: 172.28.200.101
        • Netmask: 25.255.255.0
        • Gateway: 172.28.200.2
      • DNS Server: 8.8.8.8
  • Host Name: develop → (Done)

2) Time & Date 설정

  • Region: Asia
  • Time: Seoul
  • Automatic date & time → 체크 → (Done)

3) Software Selection 설정

  • Base Environment → Server → (Done)

4) Installation Destination 설정

  • Storage Configuration → Custom 선택 → (Done) → Manual Partitioning
    • Click here to create them automatically 클릭
    • → (Done) → Accept Changes

5) Root Account 설정

  • Enable root account 클릭 → (패스워드 입력) → Allow root SSH login with password 체크

6) Begin Installation

  • 설치 진행 대기 → Reboot System 클릭
