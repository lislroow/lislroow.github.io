intellij

  • ctrl+shift+T: 검색 (in classes)
  • ctrl+shift+A: 검색 (in actions, intellij 의 기능을 검색, e.g, show lines)
  • ctrl+shift+space: javadoc (dbl-click 시 우측 탭 열림)
  • ctrl+alt+shift+N: 검색 (in symbols, 메소드 혹은 변수를 검색)
  • ctrl+shift+R: 검색 (in files)
  • alt+shift+↑: 이전 항목까지 선택 (↓ 다음 항목까지 선택)
  • ctrl+/: 주석(//, 단일행 주석)
  • ctrl+shift+/: 주석 (/* .. */, 여러행 주석)
  • ctrl+alt+↓: 현재 행을 다음행에 복사
  • alt+↑: 현재 행을 위로 이동 (↓ 아래로 이동, 메소드일 경우 메소드가 전체 이동함)
  • ctrl+-: collapse (+ expand, *: expand all)
