$ tar --help                                   
사용법: tar [<옵션>...] [<파일>]...
GNU 'tar' saves many files together into a single tape or disk archive, and can
restore individual files from the archive.

Examples:
  tar -cf archive.tar foo bar  # Create archive.tar from files foo and bar.
  tar -tvf archive.tar         # List all files in archive.tar verbosely.
  tar -xf archive.tar          # Extract all files from archive.tar.

 로컬 파일 이름 선택:

  -C, --directory=<디렉터리>   <디렉터리>로 전환
      --exclude=<패턴>     주어진 <패턴> 값에 해당하는 파일
                             제외
      --exclude-ignore=<파일>   <파일>이 있다면 각 디렉터리에
                             대한 제외 패턴을 <파일>에서 읽기
      --exclude-ignore-recursive=<파일>
                             <파일>이 있다면 각 디렉터리 및 하위
                             디렉터리에 대한 제외 패턴을 <파일>
                             에서 읽기
  -X, --exclude-from=<파일>   <파일> 목록에 있는 패턴 제외

 주 동작 모드:

  -d, --diff, --compare      아카이브 및 파일 시스템 차이점
  -r, --append               아카이브 마지막에 파일 추가
  -u, --update               아카이브 내 사본보다 최신인 파일만
                             추가

 압축 옵션:

  -a, --auto-compress        압축 프로그램 사용을 결정하는
                             아카이브 후위 확장자 사용
  -I, --use-compress-program=<프로그램>
                             <프로그램>으로 필터링합니다(-d
                             옵션을 받아들여야 함)
  -j, --bzip2                bzip2(으)로 아카이브를 필터링합니다
  -J, --xz                   xz(으)로 아카이브를 필터링합니다
      --lzip                 lzip(으)로 아카이브를 필터링합니다
      --lzma                 xz --format=lzma(으)로 아카이브를
                             필터링합니다
      --lzop                 lzop(으)로 아카이브를 필터링합니다
      --no-auto-compress     압축 프로그램 사용을 결정하는
                             아카이브 후위 확장자 사용 안함
  -z, --gzip, --gunzip, --ungzip   gzip(으)로 아카이브를
                             필터링합니다
  -Z, --compress, --uncompress   compress(으)로 아카이브를
                             필터링합니다
