key 생성

ssh-keygen 인자 -t rsa 는 rsa 타입으로 생성 입니다. key 생성 경로는 home 디렉토리의 .ssh 디렉토리에 생성이 됩니다. passphrase 는 key 의 비밀번호 입니다.

key 생성이 되면 .ssh 디렉토리에는 id_rsa(개인키), id_rsa.pub(공개키) 2개의 파일이 있습니다.

공개키인 id_rsa.pub 파일의 내용은 SSH로 접속하려는 대상 서버의 시스템계정 홈디렉토리의 .ssh/authorized_keys 파일에 추가하여 사용됩니다.
